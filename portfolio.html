Project map

A Generic & Dynamic solution for making software architecture on a web browser without using UML diagrams. This alternative to UML allows users to create subdomains, entities, value types, and enums, and even edit/add primitive types. Users can model classes add references between them, and distinguish arrayed properties, identify broken properties, and see deleted references. It includes other features, such as undo, redo, and up/download JSON files. The project, which was created using Vue.js, began as a prototype, but has been completed with a minimum set of features and deployed on Heroku server for testing.

SBRA Proceeding Processing

A government project for the Serbian Business Registers Agency (SBRA), undertaken to allow SBRA employees to manage proceedings related to health institutions. The technology stack used on this project included ASP.NET Core, RabbitMQ, Angular, and SQL. The solution assigned states to various procedural steps and involved complex business rules.
